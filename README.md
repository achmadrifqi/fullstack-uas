## How To Run

1. Clone repository
console
https://github.com/achmadrifqi/fullstack-uas.git
 
3. Copy file .env.example dan rename menjadi .env
4. Buka project di terminal dan ketik command
console
composer install
 
5. Ketik command 
console
php artisan key:generate
 
6. Ketik command 
console
php artisan migrate
 
7. Ketik command 
console
php artisan db:seed
 
8. Ketik command 
console
php artisan serve
